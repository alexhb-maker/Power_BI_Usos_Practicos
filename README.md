## 🎯 Sobre este repositorio

Este repositorio recoge una serie de **proyectos prácticos desarrollados en Power BI**, diseñados para simular escenarios reales de empresa en el ámbito financiero y de control de gestión.

El objetivo es construir una **cartera de proyectos orientada a negocio**, donde los datos se transforman en información útil para la toma de decisiones.

---

## 🧠 Enfoque

Este no es un repositorio de aprendizaje básico.

Cada proyecto está planteado como un **caso de uso real**, abordando problemas habituales en departamentos financieros:

- 🔹 Control y asignación de costes  
- 🔹 Relación entre estructura contable y operativa  
- 🔹 Análisis de información financiera  
- 🔹 Modelado de datos aplicado a reporting  
- 🔹 Visualización orientada a negocio  

El foco no está solo en la herramienta, sino en **la lógica empresarial que hay detrás del dato**.
